Javascript component framework with a twist

Get started Github

Simple

Build rich, interactive websites with ease. You are worried that your only front-end experience is basic jQuery? Worry not, it's enough to get you started with Strudel.

Declarative

Tired of all the boilerplate your front-end inventory requires? Thanks to Strudel decorators reduce it to minimum and have fun coding with ES2015 and beyond.

Lightweight

Don't worry about your website payload. Strudel makes sure users's browsers are not overwhelmed with amount of Javascript to parse as it's only ~5kb gzipped.
